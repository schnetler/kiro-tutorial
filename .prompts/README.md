## How to Use

1. Open Kiro chat panel
2. Click the **prompts icon** or use `/prompt` command
3. Select from the custom prompts above
4. Follow the interactive guidance

## Customization

Feel free to modify these prompts or create your own!

Prompt file format:
```markdown
---
name: Prompt Name
description: Brief description
---

Your prompt content here...
```

Save as `.prompts/your-prompt.prompt.md`
